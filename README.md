<h1>KdeStyleKeeper</h1>

<h2>About</h2>
KdeStyleKeeper is a simple application that is designed to help you make a quick backup of kde plasma styles with just a click of a button
 <h2>Functions</h2>
 App back up following files:<br>
 
 <ul><br>
 <li>Style config files</li>
 <li>Icons</li>
 <li>Plasmoid</li>
 <li>Look and feel folder</li>
 <li>Desktop theme</li>
 </ul>
 App does not back up following files:
 <ul>
   <li>Will be updated</li>
 </ul>
